# KROK — Free Civic Space for Belarusians

An independent civic platform providing safe, private space 
for free expression — outside regime control.

**Stack:** PHP/CodeIgniter 4, Rust (WebSocket), MariaDB

**Status:** Approaching public launch. Full codebase is 
maintained privately for operational security reasons — 
the platform operates under active threat from Belarusian 
state security services. Code review available upon request 
via secure channel.
